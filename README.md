# 2D Pac-Man Game (C, Allegro)

This project is a custom implementation of a 2D Pac-Man-style game developed using C and the Allegro graphics library.

## Features

* Player movement and game controls
* Enemy behavior and interaction
* Collision detection system
* Score tracking system
* Custom maps and gameplay features
* Additional mechanics such as cheat codes

## Technologies

* C
* Allegro Graphics Library

## Description

The game recreates core Pac-Man mechanics while introducing custom-designed maps and gameplay elements. The project focuses on implementing game logic, handling real-time interactions, and managing graphical rendering using Allegro.
